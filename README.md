## 简介
Hearthbot 是一款炉石传说 QQ 聊天机器人。基于 [nonebot2](https://github.com/nonebot/nonebot2) 开发，可与 [go-cqhttp](https://github.com/Mrs4s/go-cqhttp) 搭配使用。具有查询炉石传说卡牌、生成卡组图片等功能。
